Resumen del Proyecto

He creado una APP que permita gestionar presupuestos de forma sencilla y rápida. Los presupuestos se pueden solicitar y seguidamente aparecen en la lista inferior. Se pueden ordenar por nombre, fecha, importe y filtrar por nombre (escrito).

La APP no tiene un Backend, por lo que los datos se guardan en el LocalStorage del navegador y al refrescar la web, se borran. Lo correcto para un caso real, seria crear un Backend con una base de datos, para que los datos no se perdieran al refrescar la web.

La idea principal era crear esta APP para asentar conceptos de Angular y Typescript, ya que es un framework muy potente y versátil. Puede pasar informacion entre componentes y crear una SPA (Single Page Application) que permite navegar entre diferentes vistas sin recargar la página.

La APP está desarrollada en Angular (Typescript), HTML, Bootstrap y CSS. Está desplegada en Vercel.
